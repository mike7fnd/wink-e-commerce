# **App Name**: E-MOORM

## Core Features:

- Product Catalog: Display a catalog of products with images, descriptions, and prices.
- Product Filtering: Allow users to filter products based on categories, brands, price ranges, and other relevant attributes.  Show a "show more" option for the brands filter.
- Responsive Layout: Ensure the website layout is fluid and adapts seamlessly to different screen sizes (desktop, tablet, mobile).
- Add to Wishlist: Enable users to save the items on the wishlist
- Product Search: Allow users to search by item name, keywords, or descriptions. Implement basic typo correction for improved search accuracy.

## Style Guidelines:

- Primary color: Light, desaturated gray-blue (#D1D5DB), providing a neutral background to emphasize the products.
- Background color: Very light gray (#F9FAFB), close to white, for a clean and minimalist aesthetic.
- Accent color: Pale lavender (#E5E7EB) to highlight interactive elements like buttons and links.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, prioritizing clean legibility for product information and navigational elements.
- Use a grid-based layout to arrange product listings, filters, and other content, promoting a clean and consistent design.
- Use minimalist, line-based icons for navigation, filtering, and wishlist, for a clean and modern feel.
- Subtle animations to provide feedback upon product interactions such as hover effects on product cards and a loading animation on product filters
